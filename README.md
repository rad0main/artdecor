# ArtDecor — Каталог скинали и стеклянных изделий

> **Дата актуализации:** 2026-06-24  
> **Статус:** Разработка (Laravel 11 + Filament v3)

---

## О проекте

**ArtDecor** — это современный веб-сайт для компании, занимающейся производством и продажей скинали (стеклянных фартуков для кухни), фотопечати на стекле, декоративных панно, перегородок и сопутствующих изделий.

Проект представляет собой **кастомный движок**, заменяющий устаревшую WordPress-установку (тема Olins + 23 плагина). Основная цель — получить высокопроизводительный, безопасный и легко поддерживаемый сайт с удобной админ-панелью и интерактивным инструментом примерки.

### Бизнес-сущности

| Сущность | Описание |
|----------|----------|
| **Скинали** | Стеклянный фартук для кухни с фотопечатью |
| **Панно из стекла** | Декоративные настенные стеклянные панно |
| **Модульное панно** | Модульные стеклянные композиции |
| **Перегородки из стекла** | Интерьерные перегородки |
| **Стеклянные потолки** | Потолочные конструкции из стекла |
| **Фотоплитка** | Керамическая плитка с фотопечатью |

## Функциональные возможности

### Каталог изображений
- Фильтрация по **11 категориям** (Абстракция, Архитектура, Водопады, Города, Дизайн, Еда, Животные, Море, Природа, Текстуры, Цветы)
- Фильтрация по **13 цветам** (Красный, Оранжевый, Жёлтый, Зелёный, Синий, Тёмно-синий, Фиолетовый, Розовый, Коричневый, Бежевый, Белый, Серый, Чёрный)
- Совместная фильтрация (категория + цвет)
- Бесконечная загрузка (infinite scroll) по 12 элементов
- Избранное (через session_id для гостей)

### Онлайн-примерка
- SVG-схема кухонного гарнитура
- Выбор цвета верхних и нижних фасадов
- Подстановка изображения из каталога в зону фартука
- Отображение артикула
- Кнопки «В избранное» и «Заказать»

### Админ-панель (Filament)
- Управление каталогом (CRUD + массовая загрузка)
- Управление портфолио
- Управление услугами
- Управление заказами (статусы, фильтры, экспорт)
- Управление слайдером
- Управление категориями и цветами
- Настройки сайта (контакты, интеграции, SEO)
- Управление пользователями

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone <repo-url> artdecor_new
cd artdecor_new

# 2. Установить зависимости
composer install
npm install

# 3. Настроить окружение
cp .env.example .env
php artisan key:generate

# 4. Настроить БД в .env (PostgreSQL)
# DB_CONNECTION=pgsql
# DB_HOST=127.0.0.1
# DB_PORT=5432
# DB_DATABASE=artdecor
# DB_USERNAME=artdecor
# DB_PASSWORD=secret

# 5. Запустить миграции
php artisan migrate

# 6. Запустить dev-сервер
php artisan serve
npm run dev
```
